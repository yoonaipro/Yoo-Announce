# YooAnnounce

**YooAnnounce** – Minecraft plugin for automatic chat announcements. Includes ready-to-use `.jar` and configuration files.

## Installation
1. Download the latest `.jar` from [Releases](https://github.com/yoonaipro/Yoo-Announce/releases).
2. Place the `.jar` file in your server's `plugins` folder.
3. Start the server.
4. Edit `config.yml` (and other config files if needed) to customize announcements.
5. Restart the server or reload the plugin to apply changes.


## Configuration
* All configuration files are located in the plugin folder.
* `config.yml` – main plugin settings (messages, intervals, etc.)


## Reporting Bugs
* If you encounter issues or have suggestions, please create an [Issue](https://github.com/yoonaipro/Yoo-Announce/issues).
* Provide as much detail as possible (Minecraft version, plugin version, error messages).
